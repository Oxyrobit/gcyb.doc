## Modes d'infection courants

| Mode d'Infection | Explication |
| :--- | :--- |
| **Pièces jointes infectées** | Dissimuler dans des pièces jointes de courrier élétronique |
| **Liens malveillants** | Redirection de l'utilisateur vers un site de **phishing** |
| **Téléchargement de logiciels infectés** | Drivers sur les sites non officiel |
| **Exploitation de vulnérabilités du système** | Outils automatisés pour scanner les systèmes à la recherche de vulnérabilités |
| **Supports amovibles infectés** | Le *malware* est stocké sur un support physique (**Clé USB, disque externe**) |
| **Codes QR malveillants** (QR Code) | Le code **QR** redirige l'utilisateur vers un lien de *phishing* |
| **Attaque par la chaîne d'approvisionnement (Supply Chain Attack)** | Compromettre un **prestataire, fournisseur** ou éditeur de logiciel de confiance pour y insérer un code malveillant qui sera ensuite distribué à la cible principale. |


## Stratégies de Protection

| Stratégie | Explication / Action |
| :--- | :--- |
| **Utilisez des logiciels de sécurité** | Installez et maintenez à jour un **antivirus** et un **pare-feu** (firewall) pour la défense du système. |
| **Gardez votre système à jour** | Appliquez régulièrement les mises à jour pour corriger les failles de sécurité. |
| **Méfiez-vous des téléchargements** | **Vérifiez toujours la source** avant de télécharger un fichier ou un logiciel pour éviter les malwares. |
| **Utilisez des mots de passe forts** | Choisissez des mots de passe complexes et uniques pour sécuriser vos comptes. |
| **Faites des sauvegardes régulières** | Assurez la résilience en sauvegardant périodiquement vos données importantes. |
