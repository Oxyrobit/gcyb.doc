:::info
L'analyse de code malveillant est une activité essentielle de la cyberdéfense
:::

## Comprendre le fonctionnement du code malveillant

- Son type (Magic number)
- Son objectif
- Le moyen de propagation
- Son origine
- Est-il déjà connu ?
- Son impact sur le reseau, système/ordinateur

## Méthodes d'analyses
- Analyse statique
    - Analyse de code **sans l'éxécuter**
_ Analyse dynamique
    - Observer le code **exécuter** sans un environement contrôler
- Retro-conception
