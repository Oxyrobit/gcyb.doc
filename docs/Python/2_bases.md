# Bases et Installation

## Description
Python est un interpréteur, c'est un programme qui va prendre du un fichier contenant du texte (normalement en extension `.py`) et essayer d'interpréter le texte dans le langage qu'il connait (le Python).

Si le texte correspond au Python, il execute les commandes (le texte) indiqué.<br/>
S'il ne comprent pas un truc, il va renvoyer une erreur ou faire quelque chose d'innatendu.

## Installation
Il y a deux options:
1. Installer l'interpréteur 
2. Installer un IDE (**Recommender**)

#### Installer l'interpréteur:

Le site de python https://www.python.org/downloads/ permet de télécharcher le programme permetant d'installer l'interpréteur Python d'une **version** spécifique. <br/>

Cette histoire de **version** de Python est très importante pour certain project complexe, car certain package on besoin d'un version specifique pour fonctionner. Il n'y a malheureusement pas forcément de retro-compatibilité en utilisant une version plus avancé de Python...

#### Installer un IDE:
Un IDE est un environement de dévelopement avec plusieur option d'aide au development (débuggage, auto-completion, AI, documentation intégré...).

Le plus courant serait d'utiliser PyCharm:<br/>
https://www.jetbrains.com/fr-fr/pycharm/download/?section=windows
