AD s’appuie sur 3 protocoles :

    - **KERBEROS** = authentification réseau.

    - **DNS** = Permet de traduire un nom d’hôte en adresse IP.

    - **LDAP** = interrogation et modification des services annuaire.

Cet annuaire se situe sous `C:\Windows\NTDS\ntds.dit`.
